# cc26xx-cc13xx OTA Bootloader Example

This directory contains example code for OTA (Over-the-Air) bootloader functionality for TI CC26xx/CC13xx devices, adapted from https://github.com/msolters/contiki/tree/master/examples/cc26xx-cc13xx-ota-bootloader.

## Contents
- `ota-bootloader.c`: Main OTA bootloader logic
- `ota-bootloader.h`: Header for OTA bootloader
- `Makefile`: Build instructions
- `README.md`: This file

## Usage
Refer to the source files for implementation details. Integrate with your Contiki build system as needed.
